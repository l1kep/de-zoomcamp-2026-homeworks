# How to start
- Init bruin project zoomcamp
```bash
bruin init zoomcamp
```
- Setup bruin mcp-server. If you use windows you need to know bruin path and paste it into `command`
```yml
"bruin": {
    "command": "C:\\Users\\1\\.local\\bin\\bruin.exe",
    "args": [
    "mcp"
    ],
    "disabled": false,
    "autoApprove": [
    "bruin_get_overview",
    "bruin_get_docs_tree",
    "bruin_get_doc_content"
    ]
}
```
- Use your agent and prompt in `zoomcamp/README_bruin.md` to init project
